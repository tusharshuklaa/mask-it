FAQs

What is Mask It?

Mask It is a browser extension that helps you mask the content on the webpages. It can be used to hide sensitive information, block ads, or simply to have fun.

How do I mask an element?

You can mask an element by clicking on the Mask It extension icon in the browser and then clicking on the element you want to mask. You can also use the keyboard shortcut Alt/Option + K to mask an element.

Why sometimes ADs are not masked?

Ads on many websites are very dynamic in nature and it is quite tricky to identify them. We are working on improving the masking behavior for such elements.

I'm facing some issues. How do I report them?

You can report any issues or bugs on our Github repository. We will try to resolve them as soon as possible.

How do I unmask an element?

You can unmask an element bly clicking on the Unmask button in the extension popup (top right corenr in browser). You can also use the keyboard shortcut Alt/Option + L to unmask the last element.
